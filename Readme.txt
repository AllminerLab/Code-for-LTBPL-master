Some tips to use this package code:

1. Open run.m to run LTBPL.
2. You can replace the dataset in the experiment as you want.

The tunable parameters about different datasets are listed as follows:
1.yale                alpha=0.1 ,beta=10 ,omega=[0.1,4,10]
2.ORL                alpha=0.01 ,beta=4 ,omega=[1,10,10]
3.COIL-20          alpha=0.1 ,beta=10 ,omega=[1,10,10] (This dataset can be found in release)
4.UCI                 alpha=0.003 ,beta=8 ,omega=[8,10,10]
5.Caltech-101    alpha=0.3 ,beta=9 ,omega=[10,10,10,10,0.1,5]
6.Notting-Hill    alpha=1 ,beta=9 ,omega=[1,10,10] (This dataset can be found in release)
7.Hdigit             alpha=0.001 ,beta=60 ,omega=[20,50]
8.BBCSport        alpha=1 ,beta=50 ,omega=[1,10]
9.BBC4view        alpha=2 ,beta=10 ,omega=[9,10,10,10]

References:

Man-Sheng Chen, Chang-Dong Wang and Jian-Huang Lai. "Low-Rank Tensor Based Proximity Learning for Multi-View Clustering", TKDE2023
